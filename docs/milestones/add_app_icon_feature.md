# Feature Request: Add Application Icon (PNG, ICO, ICNS) and Integrate into UI and Build Process

## Summary
Generate and integrate a unified application icon for Vector Dataset Toolkit using the source image located at:

https://github.com/zeevbensender/vector-dataset-toolkit/blob/main/docs/milestones/vector_dataset_tool.png

This feature includes generating all necessary icon formats (PNG, ICO, ICNS, SVG), organizing icons using best practices, embedding icons through Qt Resource System, and applying them to both runtime (window/taskbar) and packaged builds (PyInstaller).

---

# Goals
1. Convert the provided PNG icon into a complete icon set for all platforms:
   - `.png` (source + runtime)
   - `.ico` (Windows executable + taskbar)
   - `.icns` (macOS bundle compatibility)
   - `.svg` (optional, for future UI scaling)

2. Store icons in a structured, future-proof location:
   ```
   resources/icons/
   ```

3. Integrate icons using Qt’s Resource System (`.qrc` file) for compatibility with PyInstaller and relative paths.

4. Update application startup code to use the new icon with:
   - `QApplication.setWindowIcon()`
   - Icons inside `.desktop` files (Linux)
   - Icons inside PyInstaller bundles

5. Provide an optional Linux desktop entry installer script.

---

# Requirements

## 1. Generate Full Icon Set
Create a Python tool at:

```
tools/generate_icons.py
```

Functionality:
- Download or load the original PNG icon.
- Automatically generate multi-resolution PNGs:
  - 16×16, 32×32, 48×48, 64×64, 128×128, 256×256, 512×512.
- Build a Windows-compatible ICO file with all resolutions bundled.
- Build macOS-compatible ICNS file.
- Optionally generate an SVG placeholder.

Use **Pillow** for resizing and multi-resolution export.

### Output:
```
resources/icons/vector_dataset_tool.png
resources/icons/vector_dataset_tool.ico
resources/icons/vector_dataset_tool.icns
resources/icons/vector_dataset_tool.svg (optional)
```

---

## 2. Add Qt Resource File
Create:

```
resources/resources.qrc
```

Example content:

```xml
<RCC>
  <qresource prefix="/icons">
    <file>icons/vector_dataset_tool.png</file>
    <file>icons/vector_dataset_tool.ico</file>
  </qresource>
</RCC>
```

Compile with:

```
pyside6-rcc resources.qrc -o resources_rc.py
```

Ensure this file is imported at application startup.

---

## 3. Integrate Icon Into Application
In the main application startup file (e.g., `app.py`):

```python
from PySide6.QtGui import QIcon

app = QApplication(sys.argv)
app.setWindowIcon(QIcon(":/icons/vector_dataset_tool.png"))
```

Window icon, taskbar icon, and dialog icons will now match.

---

## 4. Apply Icon to PyInstaller Packaging
Update build commands or `.spec` file to include:

- The `.ico` file as the executable icon
- The Qt resource module so icons load correctly at runtime

Example:

```
pyinstaller --name "VectorDatasetToolkit" --icon resources/icons/vector_dataset_tool.ico src/app.py
```

---

## 5. Optional: Linux Desktop File
Add installer script:

```
tools/install_desktop_entry.sh
```

Contents:
- Copy `.desktop` file to `~/.local/share/applications/`
- Copy PNG icon to `~/.local/share/icons/hicolor/256x256/apps/`
- Run `update-desktop-database` if available

- Full multi-resolution icon set generated successfully.
- All icons placed inside `resources/icons/`.
- Qt Resource System configured and compiled.
- Application window shows the correct icon.
- PyInstaller build uses the generated `.ico`.
